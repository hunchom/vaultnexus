# demo-vault-seeded

Sample data shipped with VaultNexus for demo and tests. Do not index alongside your real vault. The git history here is synthetic — commit dates are baked, not lived. The notes are hand-authored to exercise three wikilink-coherent topic communities (productivity, decisions, knowledge tools), sparse cross-community bridges, a single stance-shift note with three dated revisions, and five forecast-marked predictions. Run `pnpm seed:demo <output-dir>` to materialise a fresh copy with the canonical commit timeline replayed.
