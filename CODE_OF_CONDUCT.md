# Code of conduct

## TL;DR

Be excellent to each other. Don't be a jerk. If someone tells you a behavior is unwelcome, stop.

## The longer version

This project adopts the spirit of the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) without re-importing its full text. In practice:

- Disagree about technical decisions sharply if you want. Disagree about people not at all.
- Assume good faith on first interaction. Ask before assigning motive.
- Personal attacks, harassment, doxing, or persistent hostility are out of bounds.
- The maintainer's call is final on whether a behavior crosses a line.

## Reporting

Open a private security advisory at [github.com/hunchom/vaultnexus/security/advisories/new](https://github.com/hunchom/vaultnexus/security/advisories/new) — same channel as security reports. Mark the title with `[COC]` prefix.

Reports will be read within 7 days. Consequences range from a private note → public retraction request → ban, depending on severity and pattern.
